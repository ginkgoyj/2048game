# 2048

一个用 Python 编写的命令行 2048 小游戏，包含当前主版本和几个保留的迭代版本。

## 功能

- 4x4 棋盘随机生成数字块
- 支持上下左右移动与同值合并
- 统计当前得分
- 判断胜利与失败状态
- 支持重新开始和退出
- 部分版本支持本地排行榜记录
- 部分版本支持达到 `2048` 后继续挑战

## 技术栈

- Python
- 标准库：`random`、`os`、`time`

## 开发环境

- Python 3.x

## 代码结构

  - `archive/v4_with_leaderboard/game2048.py`

## 核心实现

### 棋盘与数据结构

程序使用 4x4 二维数组表示棋盘，空位用 `0` 表示。每次移动后根据空位情况随机补入新的数字块。

### 移动与合并逻辑

主版本按四个方向分别处理棋盘移动，包含：

- 提取非零数字
- 压缩到一侧
- 合并相邻相同数字
- 回写棋盘并补零

`archive/v3_refactored/game_2048.py` 对部分棋盘操作做了进一步拆分，包含 `compress_line`、`merge_line`、`reverse_line`、`transpose_board` 等函数。

### 程序流程

主循环包含以下步骤：

- 刷新棋盘显示
- 检查是否达到 `2048`
- 检查是否无路可走
- 读取用户输入
- 执行移动并更新得分
- 在有效移动后生成新的数字块

### 计分与排行榜

带排行榜的版本会在游戏结束时写入本地文本文件，并在启动或结束时读取、排序和显示历史得分。

## 如何运行

运行主版本：

```powershell
python game2048.py
```

如需查看保留版本：

```powershell
python archive/v3_refactored/game_2048.py
```
